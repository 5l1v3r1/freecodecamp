FCC Project: Image Search


Image Search is a microservice to search for images and view there URL, source URL, thumbnail and snippet in JSON format when making a GET request to


bminixhofer-image-search.herokuapp.com/imagesearch/<query>

where <query> is your search term.


You can also view the most recent searches done with this service by navigating to


bminixhofer-image-search.herokuapp.com/mostrecent
