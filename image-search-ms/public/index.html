FCC Project: URL Shortener


URL Shortener is a microservice made shorten URLs passed as a parameter
Valid url formats in this context are:

www.example.com
http://www.example.com

and a valid get request would be:


https://url-shortener-ms-bminixhofer.c9users.io/new/www.example.com

the application would shorten the url and display a JSON with the original and shortened URL like:

{ "url": "www.example.com", "short": "https://url-shortener-ms-bminixhofer.c9users.io/x" }

if an invalid URL gets passed, a JSON describing the error is shown.
