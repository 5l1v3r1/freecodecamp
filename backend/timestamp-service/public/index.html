FCC Project: Timestamp Microservice


Timestamp Microservice is a simple project to display natural and unix time when a valid date is passed as parameter.
Valid date formats in this context are:

<Month> <Date>, <Year> for example March 6, 2016
<Milliseconds since January 1, 1970> for example 1457222400000

in this example case the service would display a JSON containing natural and unix timestamps of the current date like:

{ "unix": 1457222400000, "natural": "March 6, 2016" }

if an invalid date gets passed

{ "unix": null, "natural": null }

is returned.
